## Project Name
#### simple_math

## Project Description
#### Show the usage of CCC as a simple case study

## Compilation method
#### （1） One step file compilation
```c
//Compile all files
ccc main.ccc
//Or
ccc main.ccc all
```
#### （2） Compile files step by step
```c
//Compile my_math. a file
ccc main.ccc my_math.a

//Compile the main file
ccc main.ccc main
```
